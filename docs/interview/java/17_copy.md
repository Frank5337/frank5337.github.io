# 深拷贝和浅拷贝的区别

## 题目

> 深拷贝和浅拷贝有什么区别？

## 参考答案

浅拷贝只复制对象本身，内部引用对象还是共享的；深拷贝会把引用对象也复制一份。

所以浅拷贝的问题是：

- 改外层对象通常互不影响
- 改内部引用对象，两边可能一起变

面试里可以补充实现方式，比如序列化、手动复制、拷贝构造等。
## 回答展开

- 基础题不要只给定义，最好补上原理、边界和一个容易踩坑的点。
- 如果能说明这个知识点解决什么问题、不能解决什么问题，回答会更完整。
- 最后加一句实际取舍，通常比只背术语更像真实经验。
