# 垃圾回收策略

## 题目

> 说说 JVM 的垃圾回收策略。

## 参考答案

常见垃圾回收算法有：

- 标记清除
- 复制算法
- 标记整理
- 分代收集

新生代更常用复制算法，老年代更常见标记整理。现代 JVM 越来越强调低停顿，所以像 G1、ZGC 这类回收器会更常被提到。
## 回答展开

- 基础题不要只给定义，最好补上原理、边界和一个容易踩坑的点。
- 如果能说明这个知识点解决什么问题、不能解决什么问题，回答会更完整。
- 最后加一句实际取舍，通常比只背术语更像真实经验。

## 实际场景

比如批量导出、报表聚合时，如果一次性把几十万条记录装进内存，就可能出现老年代暴涨、频繁 Full GC，最后甚至 OOM。
