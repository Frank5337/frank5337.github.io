# 什么是覆盖索引

## 题目

> 什么是覆盖索引？

## 参考答案

查询需要的字段都能从索引中直接拿到，不需要再回表，就叫覆盖索引。

它的好处是：

- 减少回表
- 提升查询效率

所以很多优化其实不是“再建一个索引”，而是把查询字段设计进已有索引里。
## 回答展开

- SQL 和索引题的关键是解释为什么快、为什么慢，而不是只背优化口诀。
- 回答时最好结合执行计划、扫描行数、回表次数和写入成本。
- 真正有经验的回答通常会强调以执行计划和真实数据分布为准。

## 实际场景

比如后台订单列表查询慢，通常要先看执行计划，再判断是不是没走索引、回表太多或深分页导致扫描成本过高。
