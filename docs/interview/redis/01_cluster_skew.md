# Cluster模式, 数据倾斜问题

## 题目

> Redis Cluster 模式下，什么是数据倾斜？怎么处理？

## 参考答案

Redis Cluster 把数据分到 16384 个槽位。数据倾斜通常来自：

- key 设计不均匀
- 热点 key 集中
- 某些业务量天然偏斜

处理方式：

- 优化 key 设计
- 热点 key 拆分
- 槽位迁移再均衡
## 回答展开

- 缓存题建议按原理、风险、治理方案三个层次展开。
- 如果涉及集群或高可用，要明确 Redis 更偏最终一致而不是强一致。
- 如果涉及缓存问题，最好把穿透、击穿、雪崩的成因和治理手段区分开。

## 实际场景

比如直播间热点数据、秒杀库存、首页爆款商品详情都可能把某个 Redis 分片打满，这时就要重点关注热点分布、脑裂风险和主从切换影响。
